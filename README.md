# 🧮 Python Calculator Project

A simple command-line calculator built using Python that performs basic arithmetic operations with error handling.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division (with zero error handling)

## ▶️ How to Run
python calculator.py
