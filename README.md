# 🧮 Python Calculator

A simple command-line calculator built using Python. It performs basic arithmetic operations with proper error handling.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division (with zero error handling)

## ▶️ How to Run
python calculator.py

## 🛠️ Tech Used
